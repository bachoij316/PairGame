# PairGame
Matching Pairs Game is a Java program designed using the Swing API to materialize GUI.
Basic logics for this game are used such as determination of two selected cards are same or not, and if they are, leave them open and count score up.  

It has 4 x 4 buttons with 8 pairs of different animal pictures. Each time when player turns two cards, the trying count goes up. The player should try to find all pairs with minimal trying counts.
